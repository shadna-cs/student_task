
مشروع ويب بسيط يربط صفحة ويب بقاعدة بيانات لتسجيل البيانات وتحديثها مباشرة بدون إعادة تحميل الصفحة.

## فكرة المشروع وتطبيقه

1. تصميم الصفحة: واجهة بسيطة تم بناؤها باستخدام HTML و CSS و JavaScript ومربوطة بالخلفية باستخدام PHP.
2. إدخال البيانات: فورم يستقبل الاسم والعمر ويقوم بحفظها مباشرة في قاعدة بيانات MySQL.
3. عرض البيانات: جدول تحت الفورم يعرض كل البيانات المخزنة في قاعدة البيانات أولاً بأول.
4. زر التبديل (Toggle): زر بجانب كل اسم في الجدول لتغيير الحالة (Status) بين 0 و 1.
5. التحديث المباشر: تتحدث الحالة في الجدول وقاعدة البيانات فوراً عند الضغط على الزر بدون الحاجة لإعادة تحميل الصفحة (باستخدام AJAX).

## خطوات التشغيل والاستضافة

* محلياً (XAMPP): نقل مجلد المشروع إلى htdocs وتفعيل Apache و MySQL من لوحة التحكم، ثم تشغيل المشروع عبر المتصفح.
* أونلاين (InfinityFree): رفع الملفات إلى مجلد htdocs الخاص بالاستضافة، وتعديل ملف الاتصال db.php ببيانات قاعدة البيانات الجديدة الخاصة بموقع InfinityFree.
